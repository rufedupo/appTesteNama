<h1>Teste Nama</h1>
<h2>Pré-requisito</h2>
<ul>
  <li>Rails 5.2.4+</li>
  <li> Ruby 2.7.2+</li>
</ul>


<h2>Instalação</h2>
<ol>
  <li>Clonando a aplicação:
    git clone https://github.com/s1du5/appTesteNama</li>
  <li>Rodar: bundle install</li>
  <li>Rodar: rake db:migrate</li>
  <li>Executar a aplicação com: rails s ou rails server</li>
</ol>
